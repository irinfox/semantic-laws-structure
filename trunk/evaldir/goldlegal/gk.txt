Статья 1052 . 
Расторжение договора простого товарищества по требованию стороны 
Наряду с основаниями , указанными в пункте 2 статьи 450 настоящего Кодекса , сторона договора простого товарищества , заключенного с указанием срока или с указанием цели в качестве отменительного условия , вправе требовать расторжения договора в отношениях между собой и остальными товарищами по уважительной причине с возмещением остальным товарищам реального ущерба , причиненного расторжением договора . 
	LegalInfo 
        { 
                Subject = сторона договора простого товарищества, заключенного с указанием срока или с указанием цели в качестве отменительного условия, 
                SubjectKey = сторона договора простого товарищества 
                Modality = Permission
		Pred = требовать 
                ObjectPlainText = расторжения договора в отношениях между собой и остальными товарищами по уважительной причине с возмещением остальным товарищам реального ущерба, причиненного расторжением договора
		Reference = пункте 2 статьи 450 настоящего Кодекса 
        }

Статья 827 . 
Ответственность клиента перед финансовым агентом 
1 . 
Если договором финансирования под уступку денежного требования не предусмотрено иное , клиент несет перед финансовым агентом ответственность за действительность денежного требования , являющегося предметом уступки . 
	ConditionClause
	{
		Antecedent = если договором финансирования под уступку денежного требования не предусмотрено иное
		ASubj = договором финансирования под уступку денежного требования
		APred = не предусмотрено
		AObj = иное
		Consequent = клиент несет перед финансовым агентом ответственность за действительность денежного требования , являющегося предметом уступки
		CSubj = клиент
		CPred = несет ответственность
		CObj = перед финансовым агентом за действительность денежного требования , являющегося предметом уступки
	}
2 . 
Денежное требование , являющееся предметом уступки , признается действительным , если клиент обладает правом на передачу денежного требования и в момент уступки этого требования ему не известны обстоятельства , вследствие которых должник вправе его не исполнять . 
	ConditionClause {}
	LegalInfo
	{
		Subject = денежное требование, являющееся предметом уступки,
		SubjectKey = денежное требование
		Pred = признается действительным
		Inversion = false
	}
	LegalInfo
	{
		Subject = клиент
		SubjectKey = клиент
		Pred = обладает
		ObjectPlainText = правом на передачу денежного требования
	}
	LegalInfo
	{
		Subject = обстоятельство
		SubjectKey = обстоятельство
		Pred = не изветсны
		ObjectPlainText = ему 
		Time = в момент уступки этого требования
	}
	LegalInfo
	{
		Subject = должник
		SubjectKey = должник
		Pred = не исполнять
		Modality = Permission
		ObjectPlainText = его
	}
3 . 
Клиент не отвечает за неисполнение или ненадлежащее исполнение должником требования , являющегося предметом уступки , в случае предъявления его финансовым агентом к исполнению , если иное не предусмотрено договором между клиентом и финансовым агентом . 
	LegalInfo
	{
                Subject = клиент 
                SubjectKey = клиент 
		Pred = не отвечает 
		ObjectPlainText = за неисполнение или ненадлежащее исполнение должником требования,являющегося предметом уступки, 
		Object1 = за неисполнение или ненадлежащее исполнение должником 
		Object2 = требования,являющегося предметом уступки, 
		Exception = если иное не предусмотрено договором между клиентом и финансовым агентом
		Condition = в случае предъявления его финансовым агентом к исполнению 
	}

Статья 684 . 
Преимущественное право нанимателя на заключение договора на новый срок 
По истечении срока договора найма жилого помещения наниматель имеет преимущественное право на заключение договора найма жилого помещения на новый срок . 
	LegalInfo
	{
		Subject = наниматель
		SubjectKey = наниматель
		Pred = имеет
		ObjectPlainText = преимущественное право на заключение договора найма жилого помещения на новый срок
		Time = По истечении срока договора найма жилого помещения
		Inversion = false
	}
Не позднее чем за три месяца до истечения срока договора найма жилого помещения наймодатель должен предложить нанимателю заключить договор на тех же или иных условиях либо предупредить нанимателя об отказе от продления договора в связи с решением не сдавать в течение не менее года жилое помещение внаем . 
	LegalInfo
	{
		Subject = наймодатель
		SubjectKey = наймодатель
		Pred = предложить заключить
		Modality = Obligation
		ObjectPlainText = нанимателю договор на тех же или иных условиях 
		Time = Не позднее чем за три месяца до истечения срока договора найма жилого помещения
	}
	LegalInfo
	{
		Pred = предупредить
		ObjectPlainText = нанимателя об отказе от продления договора в связи с решением
	}
Если наймодатель не выполнил этой обязанности , а наниматель не отказался от продления договора , договор считается продленным на тех же условиях и на тот же срок . 
	ConditionClause {}
	LegalInfo
	{
		Subject = наймодатель
		SubjectKey = наймодатель
		Pred = не выполнил
		ObjectPlainText = этой обязанности 
	}
	LegalInfo
	{
		Subject = наниматель 
		SubjectKey = наниматель 
		Pred =  не отказался
		ObjectPlainText = от продления договора 
	}
	LegalInfo
	{
		Subject = договор
		SubjectKey = договор
		Pred = считается продленным
		ObjectPlainText = на тех же условиях и на тот же срок
	}
При согласовании условий договора наниматель не вправе требовать увеличения числа лиц , постоянно с ним проживающих по договору найма жилого помещения . 
	LegalInfo
	{
		Subject = наниматель
		SubjectKey = наниматель
		Pred = требовать
		Modality = Prohibition
		ObjectPlainText = увеличения числа лиц, постоянно с ним проживающих по договору найма жилого помещения
		Condition = При согласовании условий договора
	}
Если наймодатель отказался от продления договора в связи с решением не сдавать помещение внаем , но в течение года со дня истечения срока договора с нанимателем заключил договор найма жилого помещения с другим лицом , наниматель вправе требовать признания такого договора недействительным и ( или ) возмещения убытков , причиненных отказом возобновить с ним договор . 
	ConditionClause {}
	LegalInfo
	{
		Subject = наймодатель
		SubjectKey = наймодатель
		Pred = отказался
		ObjectPlainText = от продления договора в связи с решением не сдавать помещение внаем
	}
	LegalInfo
	{
		Pred =  заключил 
		ObjectPlainText = договор найма жилого помещения с другим лицом
		Time = в течение года со дня истечения срока договора с нанимателем
	}
	LegalInfo
	{
		Subject = наниматель
		SubjectKey = наниматель
		Pred = требовать
		ObjectPlainText = признания такого договора недействительным и (или) возмещения убытков, причиненных отказом возобновить с ним договор
	}
